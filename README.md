# DSPSeedCalc
Dyson Sphere Program Seed Calculator

# Usage
* Command Line. ```DSPSeedCalc [-i filename] [-s star.csv] [-b birth.csv] [ranges...]```
* Ranges format. a-b means [a,b] (e.g. 0-999)
* Note. You need to supply either [filename] or [ranges...]
